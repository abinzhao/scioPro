## RandomColor 随机颜色

可以随机生成 16 进制或 RGB 颜色，适用于一些需要随机变化颜色的场景。

## API

| 属性名 | 描述         | 类型             | 默认值  |

| type   | 生成颜色类型 | `"#16" ， "rgb"` | `"#16"` |
